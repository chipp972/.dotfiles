# System #

*   Bios charge Master Boot (MBR) dans secteur 0 qui charge le bootloader (grub) qui charge le système. (charger = mettre en mémoire (RAM) du code)
*   BSS = variables static non initialisées
*   Mémoire virtuelle permet d'isoler les proc. Sans MMU (hardware) un proc peut accéder aux données des autres sans provoquer d'erreur (faille de sécurité)
*   strace -f main.exe pour obtenir les appels sys avec le pid
*   ltrace pour obtenir les appels de bibliotheque

Processeurs
*   Hyperthreading = simulation de multicore. Utilisation des unités de calcul (ALU) par 2 processus sur le même processeur. Mémoires L1, L2 et L3 partagées. Augmentation de perfs de 1.15-1.3 max.
*   Multicore = plusieurs puces ayant chacune leurs ALU et cache L1 mais cache L2 et L3 partagés (ou seulement L3)
*   Multiprocesseur = plusieurs processeurs complets avec caches indépendants connectés à la même mémoire.
