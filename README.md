# Description #
My configuration files and automatic installation scripts

# Utilisation #
To install:
```
git clone https://github.com/chipp972/.dotfiles
sh .dotfiles/install.sh
```
To uninstall:
```
sh .dotfiles/scripts/uninstall.sh
rm -rf .dotfiles
```
