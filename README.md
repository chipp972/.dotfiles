# Description #
My configuration files and automatic installation

# Utilisation #
To install:
```
git clone https://github.com/chipp972/.dotfiles
.dotfiles/scripts/install.sh
```
To uninstall:
```
.dotfiles/scripts/uninstall.sh
rm -rf .dotfiles
```
