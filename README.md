# DOTFILE

## Description

Configuration files and automatic installation scripts for Ubuntu 18+

| Package name           | all |
| ---------------------- | --- |
| git                    |  X  |
| git-flow               |  X  |
| neovim                 |  X  |
| vim-plug               |  X  |
| xclip                  |  X  |
| fira-code fonts        |  X  |
| htop                   |  X  |
| vscode                 |  X  |
| bash-it                |  X  |
| chromium               |  X  |

## Usage

To install:

```bash
git clone https://github.com/chipp972/.dotfiles ~
bash ~/.dotfiles/install.sh
```
